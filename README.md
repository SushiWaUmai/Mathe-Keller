# Mathe Keller

A project for students to learn math!

## Development 
### Prerequisites
- node
- yarn
### Getting started

Clone the repository
```bash
git clone https://github.com/SushiWaUmai/Mathe-Keller.git
cd ./Mathe-Keller
```

Install dependencies and start developing
```bash
yarn install
yarn dev
```
